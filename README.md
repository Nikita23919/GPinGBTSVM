# GPinGBTSVM
Granular ball twin support vector machine with generalized pinball loss
gen_ball.py: Generates granular balls
This part of the code is adapted from “Granular Ball Support Vector Machine” (Xia et al., 2022). The implementation is simplified and documented for clarity, and may differ from the version used in the paper.
